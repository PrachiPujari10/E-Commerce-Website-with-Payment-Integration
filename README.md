# 🛍️ E-Commerce Website with Payment Integration

A modern **Full Stack MERN E-Commerce Website** built using **React.js, Node.js, Express.js, and MongoDB**. This project provides a complete online shopping experience with secure authentication, product management, an admin dashboard, shopping cart, order management, and multiple payment options including **Stripe**, **Razorpay**, and **Cash on Delivery (COD)**.


---

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/PrachiPujari10/E-Commerce-Website-with-Payment-Integration.git
```

### Install Dependencies

Frontend

```bash
cd frontend
npm install
```

Backend

```bash
cd backend
npm install
```

Admin

```bash
cd admin
npm install
```

---

# ▶️ Run Project

Backend

```bash
npm run server
```

Frontend

```bash
npm run dev
```

Admin

```bash
npm run dev
```

---

# 🔑 Environment Variables

Backend `.env`

```env
MONGODB_URI=

JWT_SECRET=

CLOUDINARY_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_SECRET_KEY=

STRIPE_SECRET_KEY=

RAZORPAY_KEY_ID=

RAZORPAY_KEY_SECRET=
```

Frontend `.env`

```env
VITE_BACKEND_URL=

VITE_RAZORPAY_KEY_ID=
```

Admin `.env`

```env
VITE_BACKEND_URL=
```
